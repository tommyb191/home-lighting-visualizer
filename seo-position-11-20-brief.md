# Website SEO Update — Implementation Brief

**Site:** phantomsound.com
**Prepared:** 2026-06-12
**For:** Web editor / coworker implementing the changes
**Goal:** Push five pages that currently rank on **page 2 of Google (positions 11–20)** onto page 1 by making each page clearly match the search terms people are already using to find it.

---

## How this was chosen

Pulled from Google Search Console (Performance → last 3 months → Web). These five pages get lots of Google impressions but sit just off page one. For each, we identified the exact search term (query) it should target, then rewrote the page to match.

You do **not** need to understand SEO to do this. For each page below, make these five edits:

1. **Title tag** — the page's SEO title (in WordPress this is usually the "SEO Title" field in Yoast/Rank Math, *not* the visible heading).
2. **Meta description** — the gray snippet under the title in Google results (the "Meta Description" field in Yoast/Rank Math). Doesn't affect ranking, but improves click-through.
3. **H1** — the big visible headline at the top of the page.
4. **Add a section** — paste the provided paragraph into the page body (near the top, below the hero/intro). Adjust wording so it reads naturally with what's already there; don't delete good existing content.
5. **Add 5 internal links** — inside the body text, link the exact words shown ("anchor text") to the listed URL.

**Link tips:** Use the exact anchor words shown. Make them real in-body text links, not buttons or "click here."

## After publishing (do this for each page)

1. In **Google Search Console** → top search bar → paste the page URL → press Enter → click **Request Indexing**. This tells Google to re-crawl sooner.
2. Re-check rankings in about **2–4 weeks** (GSC → Performance → filter by each page).
3. Don't change the URLs themselves — only the title, headline, body text, and links.

## Important notes for the editor

- These paragraphs are written to match the search terms. **Adapt the wording to fit each page's existing content** — don't blindly delete sections that are already there. The key is that the new H1, title, and the target phrases (e.g., "sound system installation in Cincinnati") appear naturally in the visible text.
- If any of the linked destination URLs has changed (e.g., a page was renamed), use the current URL but keep the same anchor-text wording.
- Keep it natural. The text should read like a human wrote it for customers, not stuffed with keywords.
